The core component engine of AW.

Key classes include {@link ariba.ui.aribaweb.core.AWComponent AWComponent}, {@link ariba.ui.aribaweb.core.AWRequestContext AWRequestContext}, {@link ariba.ui.aribaweb.core.AWRequest AWRequest}, {@link ariba.ui.aribaweb.core.AWResponse AWResponse}, {@link ariba.ui.aribaweb.core.AWSession AWSession}, {@link ariba.ui.aribaweb.core.AWApplication AWApplication}, and {@link ariba.ui.aribaweb.core.AWDirectAction AWDirectAction}.

Core also includes basic "elements" for control and dynamic html rendering, including {@link ariba.ui.aribaweb.core.AWIf a:If}, {@link ariba.ui.aribaweb.core.AWFor a:For}, {@link ariba.ui.aribaweb.core.AWIncludeComponent a:IncludeComponent}, {@link ariba.ui.aribaweb.core.AWGenericElement a:GenericElement}, and {@link ariba.ui.aribaweb.core.AWGenericContainer a:GenericContainer},

Finally, core contains the foundation of AW's automatic incremental/dynamic refresh capability: {@link ariba.ui.aribaweb.core.AWRefreshRegion a:RefreshRegion}. @see ariba.ui.aribaweb.html
